一。js中的函数分为四种。
1.普通函数
语法：

function 函数名(参数列表){
函数体
}

注意：
（1）js中函数的调用可以放在声明之前（放位置的时候不可以越过script标签)
（2）调用的函数的时候，任何函数都可以传任意个参数，跟形参没有关系。
（3）形参可以直接获取部分实参的值；函数中arguments对象可以获取函数调用时所有的实参。
（4）关键字参数是无效的，有效 的是位置的参数，按位置传参数。

2.匿名函数：
语法：
变量 = function (参数列表){
函数体
}
调用的时候：变量（）

有时候传入的参数需要是函数的时候，这时候就可以将匿名函数当做参数传入到这里面。

例：

3.箭头函数
语法：

(参数列表)=>{
函数体
}
调用的时候：也是直接赋给变量调用。

4.自调用函数
语法：

(function(形参列表){
函数体
})(实参列表)

解析：这个函数就是声明之后整体加括号在调用这个函数。

注意：自调用函数在书写的时候，上一条书写的内容是必须加 分号的，否则该自调用函数会报错

48-js循环结构

发表于 2020-01-05
本文字数： 464 阅读时长 ≈ 1 分钟

一.js中循环有for 循环while循环。
1.for循环
（1）c语言中for循环
语法：for(表达式1;表达式2;表达式3){
循环体
}

上面这个就相当于：

表达式1
for(;表达式2;){
循环体
表达式3
}

相当于while循环中的：

表达式1
while 表达式2:
循环体
表达式3
####用于：

for(给变量赋初值;循环条件;修改变量的值){
循环体
}

（2）for -in 循环：
语法：
for(变量 in 序列){
循环体
}
解析： 让变量去序列中取值一个一个的取(取的是下标/属性名)，取完为止，每取一个值执行一次循环体。
（3） for-of 循环：
语法：
for(变量 of 序列){
循环体
}
解析：让变量去序列中取值一个一个的取(取的值)，取完为止，每取一个值执行一次循环体。

注意：for of 是无法遍历对象中的值的，（python中的字典），还是的用for in 取到属性 再通过属性得到值。

2.while 循环
（1）while循环
语法：
while(条件语句){
循环体
}
（2）do-while循环
语法：
do{
循环体
}while(条件语句)
解析：先执行循环体的语句再判断。

47-js分支结构

发表于 2020-01-05
本文字数： 398 阅读时长 ≈ 1 分钟

一。js中有两种分支结构：if switch
1.if结构：
if(条件语句){
满足条件需要执行的代码
}
=======================
if(条件语句){
满足条件要执行的代码
}
else{
不满足条件要执行的代码
}
=======================
if(条件语句1){
代码段1
}
else if(条件语句2){
代码段2
}
else if(条件语句3){
代码段3
}
…
else{
代码段N
}
2..switch语法

swicth(表达式){
case 值1:{
表达式1
}
case 值2:{
表达式2
}
…
default:{
表达式N
}
}
解析： 先计算表达式的值，然后让这个值依次和case后面的每个值进行比较，哪个case后面的值和表达式的值相等，就将这个case 作为入口，依次执行后面的表达式，直接执行完或者执行过程中break为止。如果没有哪个case的值和表达式的值相等，那么就会将default作为入口。

46-js运算符

一。js支持运算符，数学，比较，逻辑，赋值运算。
1.数学运算符：: +,-,,/,%,*, ++, –
和数学的功能一样； 没有整除对应的运算符，会通过Math模块提供的方法来支持取整的操作
注意： 其中 ++ 自加1 –自减1
num=10
num++ num=num+1
++num num=num+1

num– num=num-1
–num num=num-1

其中的符号位置的区别在于：
赋值的时候不一样。
num1=10

num2=num1++ (这个表示先把num1赋值num2 然后自己再加一)

num2=++num1 (这个表示先把 自己再加一，然后再赋值给num2)

2.比较运算符：.比较运算符:>, <, ==, !=, ===, !==
结果都是布尔值
判断大小和python一样
== 判断相等的时候，如果数据类型不同会先转换成相同的数据类型，然后再判断是否相等。

===判断相等的时候，如果数据类型不同结果直接是false。

3.逻辑运算符: &&(逻辑与), ||(逻辑或), !(逻辑非)

4..赋值运算符: =, +=, -=, =, /=, %=, *=
和python的一样。

a.没有声明在{}中，从声明开始任何位置都可以用
b.如果在{}中声明，就只能在{}中使用
c.可以修改变量的值
####4. const 变量名 = 值
a.变量的值不能修改

1.盒子模型
每个可见的标签其实都是由content、padding、border和margin四个部分组成；
其中content、padding、border是可见的，margin是不可见的，但是占位置

2.控制盒子模型
(1）content - 设置标签的宽度和高度其实就是在设置content的大小;
标签中的内容或者子标签都是显示或者添加到content上的;
设置标签的背景会作用于content
( 2)padding(内边距) - padding分四个方向；这四个方向可以单独控制(大小);
设置标签的背景会作用于padding

( 3)border (边框) - border分四个方向；每个方向可以单独控制(大小、样式和颜色);

(4)margin(外边距) - margin分四个方向; 每个方向可以单独控制(大小)
不可见，但是占位置

1)一起设置(同时给四个方向赋值)

padding-top
padding-bottom

同时设置四个方向的圆角*/
